# GRDH - guide RNA design helper

**Программа для поиска, подбора и анализа направляющих РНК (нРНК) для системы CRISPR/Cas9.**

## Возможности

### Источники входной последовательности

- `.fasta`-файл с опциональным указанием границ региона
- Собственная последовательность, введённая вручную
- ID локуса в базе NCBI nucleotide с опциональными границами
- Выдача инструмента CRISPOR (`.xls`) - анализируется наравне с локальным поиском

### Скоринг и оценка нРНК

- **Doench 16 (Azimuth)** - эффективность разреза, модель извлечена из оригинального Azimuth и воспроизводит его результаты
- **OTS (Our Total Score)** - собственный балл, учитывающий:
  - GC-состав
  - олигонуклеотидные повторы
  - нуклеотиды в ключевых позициях спейсера (3, 14, 16, 18, 20)
  - наличие GCC на участке 16–20
  - доступность спейсера во вторичной структуре (положения 18–20 и 51–53)
  - наличие 8 и более спаренных нуклеотидов подряд и более 12 спаренных всего
  - N-позицию PAM
- **Поиск сайтов рестрикции** вблизи точки разреза с подсчётом их количества в радиусе 100, 250 и 500 п.н.

### Вторичная структура нРНК

- Точечно-скобочная структура и MFE рассчитываются библиотекой [ViennaRNA](https://www.tbi.univie.ac.at/RNA/)
- Отрисовка 2D-структуры в `.png` - собственная реализация
- Расчёт доступности ключевых позиций спейсера
- Сохранение изображения структуры для каждой нРНК

### Интерактивная геномная карта

- Координатный зум
- Стрелки, отображающие направление нРНК (sense/antisense), масштабируемые вместе с зумом
- Красная линия разреза Cas9, PAM вынесен рядом со спейсером
- Автоматическая раскладка перекрывающихся нРНК по дорожкам
- Всплывающие подсказки с параметрами нРНК при наведении

### Работа с таблицей нРНК

- Двусторонняя синхронизация: выбор нРНК на карте выделяет строку в таблице и наоборот
- Удаление нРНК из таблицы с автоматическим обновлением карты
- Подсветка ячеек по критериям качества: Doench < 50, T в N-позиции PAM, олиго-T, GCC на 16–20, более 8 спаренных подряд, более 12 спаренных всего
- Сортировка по любому столбцу

### Режим подбора пары нРНК

- Выбор нРНК-якоря с автоматической фильтрацией остальных по расстоянию между точками разреза (50–2000 п.н.)
- Подходит для экспериментов с парными никазами и делеций
- Кнопка сброса фильтра для возврата к полному списку

### Экспорт

- Экспорт оставленных в таблице нРНК в формат `.xlsx` с изображениями структур
- Все изображения структур текущей сессии сохраняются в папке `output`

## Отличия от других инструментов

Сравнение проведено с инструментами **CRISPOR**, **CRISPR-P 2.0**, **CRISPRon**, **CRISPR DB**, **sgDesigner** и **RNAfold** (ViennaRNA).

### Уникальные возможности GRDH

| Возможность | Описание |
|---|---|
| Подбор пары нРНК | Фильтрация по расстоянию между точками разреза (50–2000 п.н.) - недоступен ни в одном из перечисленных инструментов |
| Двусторонняя синхронизация карта↔таблица | Выбор нРНК на карте выделяет строку в таблице и наоборот, удаление из таблицы обновляет карту |
| Комбинация структуры + карты + скоринга | Вторичная структура, интерактивная карта и собственный скоринг в одном инструменте |
| Импорт выдачи CRISPOR | Анализ `.xls`-выдачи CRISPOR с последующим построением вторичной структуры, расчётом OTS и отображением на карте |
| Локальная работа с GUI | CRISPOR и CRISPR-P 2.0 работают только через браузер, sgDesigner и CRISPRon - только через командную строку |

### Сильные стороны

| Возможность | GRDH | CRISPOR | CRISPR-P 2.0 | CRISPRon | CRISPR DB | sgDesigner | RNAfold |
|---|---|---|---|---|---|---|---|
| Doench 16 (Azimuth) | + | + | - (Doench 14) | - | - | - | - |
| Вторичная структура нРНК | + своя отрисовка | - | + визуализатор ViennaRNA (точечно-скобочная структура) | ⚠️ энергия фолдинга без отрисовки | - | - | + (только структура) |
| Сайты рестрикции | + 100/250/500 п.н. | + | + | - | - | - | - |
| Интерактивная карта | + зум до нуклеотидов | - таблица | + | - | - | - | - |
| Off-target | - (в планах) | + MIT + CFD | + | + CRISPRoff | - | - | - |

### Слабые стороны

- **Off-target анализ не реализован** (в планах). CRISPOR использует MIT и CFD, CRISPR-P 2.0 - собственный скоринг. Для оценки off-target в данной версии рекомендуется использовать CRISPOR, выдачу которого GRDH может импортировать
- **Поддерживается только SpCas9.** CRISPR-P 2.0 и CRISPOR поддерживают другие Cas-системы (в планах)
- **Нет расчёта праймеров для клонирования** - есть у CRISPOR
- **Нет оценки микрогомологии** - есть у CRISPR-P 2.0
- **OTS - эвристический балл**, не обученный на экспериментальных данных. CRISPRon и CRISPR DB вероятно точнее предсказывают эффективность разреза
- **Нет привязки к геному** - GRDH работает с произвольной последовательностью. CRISPOR и CRISPR-P 2.0 ищут нРНК в контексте выбранного генома

## Руководство

1. Выберите режим поиска нРНК: `.fasta` / собственная последовательность / NCBI / CRISPOR.
2. **a.** При выборе `.fasta` укажите путь до файла. При необходимости задайте границы региона.\
   **b.** При выборе собственной последовательности вставьте её в поле ввода (только ATGC, не более 5 тыс. п.н.).\
   **c.** При выборе NCBI укажите ID локуса в базе nucleotide. При необходимости задайте границы региона.\
   **d.** При выборе CRISPOR укажите путь до `.xls`-файла с результатами CRISPOR.
3. Для локального поиска длина целевой последовательности должна быть не менее 50 и не более 5 тыс. п.н. При ошибке исправьте значения границ.
4. Нажмите кнопку «Анализ». При успешном запуске появятся полоса прогресса, счётчик проанализированных нРНК и последовательность текущего спейсера.
5. После завершения анализа появятся карта и таблица с результатами.
   - Приближение карты колесом мыши позволяет увидеть отдельные нуклеотиды. Красной чертой показано место разреза Cas9, PAM отображается рядом со спейсером. Наведение на спейсер показывает его параметры.
   - Клик по нРНК на карте выделяет соответствующую строку в таблице. Выбор строки в таблице подсвечивает нРНК на карте.
6. Таблица содержит основную информацию о нРНК. Изначально она отсортирована по Doench 16, значения ниже 50 отмечаются красным.
   - Ячейки Sequence и PAM подсвечиваются при наличии: T в N-позиции PAM (NGG), более 8 спаренных нуклеотидов подряд и более 12 спаренных всего в спейсере, GCC в положениях 16–20, четырёх T подряд.
   - Можно удалить выбранные нРНК или все, отмеченные красным - карта обновится автоматически.
7. Для подбора пары нРНК нажмите кнопку «Подбор пары», затем кликните по нРНК на карте. Остаются только нРНК с точкой разреза в диапазоне 50–2000 п.н. от выбранной. Кнопка «Сбросить фильтр» возвращает полный список.
8. Экспорт в формат `.xlsx` осуществляется только для оставленных в таблице нРНК. Все изображения структур текущей сессии хранятся в папке `output`.
